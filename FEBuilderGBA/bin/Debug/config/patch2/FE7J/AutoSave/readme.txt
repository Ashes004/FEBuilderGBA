AutoSaveSwitch

ガールズにあった、オートセーブ無効の切り替えを実現するパッチです。

ガールズと同じく、地形ウィンドウの切り替えはできなくなり、
オフにすると「メニューの中断からのみ状態が保存されます」


なお、例のごとく文字やアイコンはご自分で書き換えて下さい。


--------------------------------

　・地形ウィンドウ「オン」のヘルプ指定
0xDAF0F4:05 06
　・地形ウィンドウ「オフ」のヘルプ指定
0xDAF0FC:05 06
　・地形ウィンドウの「オン」「オフ」の文字指定
0xDAF0F6:17 06
0xDAF0FE:18 06
　・地形ウィンドウのアイコン指定
0xDAF114:02

・「設定」のアイコン（圧縮）
0x43EA50
